◆ integrate()

procedure, public integral_core::ode_integrator::integrate ( )

Performs the integration.

Syntax
logical function integrate(class(ode_integrator) this, class(ode_helper) fcnobj, real(real64) x(:), real(real64) y(:), optional class(errors) err)
Parameters
[in,out]thisThe ode_integrator object.
[in,out]fcnobjThe ode_helper object containing the equations to integrate.
[in]xAn array containing the values of the independent variable at which the solution is desired. There must be at least two values in this array.
[in]yAn N-element array containing the initial conditions for each of the N ODEs.
[in,out]errAn optional output that can be used to provide an error handling mechanism. If not provided, a default error handling mechanism will be utilized. Possible errors that may be encountered are as follows.
  • INT_INVALID_INPUT_ERROR: An invalid input was supplied.
  • INT_OUT_OF_MEMORY_ERROR: There is insufficient memory available.
  • INT_LACK_OF_DEFINITION_ERROR: Occurs if no equations have been defined.
  • INT_ARRAY_SIZE_MISMATCH_ERROR: Occurs if y is not sized to match the problem as defined in fcnobj, or if the tolerance arrays are not sized to match the problem as defined in fcnobj. Notice, specific integrators may have additional errors. See the step routine of the appropriate integrator for more information.
Returns
Returns the solution in a matrix of N+1 columns. The first column contains the values of the independent variable at which the solution was computed. The remaining columns contain the solution points for each ODE.

◆ step()

procedure(ode_integrator_interface), deferred, pass, public integral_core::ode_integrator::step ( )

Takes a single integration step towards the desired point.

Syntax
logical function step(class(ode_integrator) this, class(ode_helper) fcn, real(real64) x, real(real64) y(:), real(real64) xout, real(real64) rtol(:), real(real64) atol(:), optional class(errors) err)
Parameters
[in,out]thisThe ode_integrator object.
[in,out]fcnAn ode_helper object containing the ODEs to integrate.
[in,out]xOn input, the value of the independent variable at which to start. On output, the value of the independent variable at which integration terminated.
[in,out]yOn input, the value(s) of the dependent variable(s) at the initial value given in x. On output, the value(s) of the dependent variable(s) as evaluated at the output given in x.
[in]xoutThe value of the independent variable at which the solution is desired.
[in]rtolAn array containing relative tolerance information for each ODE.
[in]atolAn array containing absolute tolerance information for each ODE.
[in,out]errAn optional argument that can be used to control the error handling behavior of the integrator.
Returns
Returns true if the integrator requests a stop; else, false, to continue as normal. A possible sitaution resulting in a true value is in the event a constraint has been satisfied.
