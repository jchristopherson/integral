Detailed Description

Defines a routine for computing a single integration step in the direction of xout.

Parameters
[in,out]thisThe ode_integrator object.
[in,out]fcnAn ode_helper object containing the ODEs to integrate.
[in,out]xOn input, the value of the independent variable at which to start. On output, the value of the independent variable at which integration terminated.
[in,out]yOn input, the value(s) of the dependent variable(s) at the initial value given in x. On output, the value(s) of the dependent variable(s) as evaluated at the output given in x.
[in]xoutThe value of the independent variable at which the solution is desired.
[in]rtolAn array containing relative tolerance information for each ODE.
[in]atolAn array containing absolute tolerance information for each ODE.
[in,out]errAn optional argument that can be used to control the error handling behavior of the integrator.
Returns
Returns true if the integrator requests a stop; else, false, to continue as normal.
